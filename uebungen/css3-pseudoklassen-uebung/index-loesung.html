CSS3 Pseudoklassen

Aufgabe

1. Alle Listenpunkte stehen nebeinander.
2. Leere Listenpunkte bekommen einen farbigen border.
3. Das 4. Listenelement soll in einer Farbe und kursiv dargestellt werden.
4. Ausser dem Letzten, bekommen alle Listenelement einen border-right.

  • Sonne

    li {
       color: #999;
       display:inline;
       padding: 0 10px ;
       border-right: 1px solid #999;
       }

    li:last-child{
       border-right:none;
       }

    li:nth-of-type(4){
       color:#f67;
       font-style:italic;
       }
    li:empty {
       border: 1px solid #f67;
       }
