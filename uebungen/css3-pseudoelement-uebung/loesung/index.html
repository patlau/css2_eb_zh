CSS3 Pseudoelemente

Aufgabe

  1. Download Übungen als zip folder & auspacken (Aufgabe ansehen)
  2. Download Iconfont von Fontawesome
  3. Lege diesen Folder in deinen Projektfolder (css3-pseudoelement-uebung).
  4. Importiere das font-awesome.css in deine bereits vorhandene CSS Datei. (style.css)
  5. CSS Datei mit CSS3 dem Pseudoelement:after ergänzen.
  6. Look mit einem :hover Effekt ergänzen.
  7. Siehe Lösung
